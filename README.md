# cta-button-js

クリックすると背景が変化し、メッセージが表示される  
シンプルなCTAボタンを作成しました。

HTML / CSS / JavaScript を使い、  
DOM操作やクラスの付け替え、簡単なアニメーション表現を学習する目的で制作しています。

## 機能
- CTAボタンをクリックすると背景が変化
- メッセージをフェードイン表示
- リセットボタンで元の状態に戻す
- 押下時に「押した感」が出るUI演出

## 使用技術
- HTML
- CSS
- JavaScript

## 学習ポイント
- `classList` を使ったスタイル切り替え
- `transform` / `transition` によるUI演出
- DOMの生成・削除
